Description
The original program is a simple console-based game of Rock, Paper, Scissors developed in C++. It allows a player to play a fixed number of rounds (1â€“10) against the computer.

What the Program Does
Asks the user how many rounds they want to play (up to 10).

In each round:

The player chooses Rock, Paper, or Scissors.

The computer randomly selects its move.

The program decides the winner of the round.

After all rounds, the final game winner is announced based on total wins.

Key Features
Fixed rounds.

Random computer choice.

Colored console output to indicate winner/loser (Windows only).

Game over screen and summary at the end.
