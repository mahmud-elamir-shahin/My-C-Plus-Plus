Math Quiz Game – C++ Project Description

This C++ program is an interactive Math Quiz Game designed to help users practice basic arithmetic operations at different difficulty levels. It generates random math questions, evaluates user answers, and provides feedback and results.

Key Features:

- Difficulty Levels:
  Users can choose from:
  - Easy (numbers 1–10)
  - Medium (10–50)
  - Hard (50–100)
  - Mix (randomly selects one of the above)

- Operation Types:  
  Users can select:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mix (random operation)

- Random Question Generation:
  The program generates a set of random math questions based on the selected level and operation.

- Answer Evaluation:
  After each question, the program checks the user's answer and gives immediate feedback:
  - ✅ Green screen for correct answers
  - ❌ Red screen and correct answer shown for wrong answers

- Final Results Summary:
  At the end of the quiz, the program displays:
  - Total number of questions
  - Number of correct and incorrect answers
  - Whether the user passed (more correct than incorrect)

- Replay Option: 
  Users can choose to play again after finishing a quiz.

Educational Purpose:

This project is ideal for beginners learning:
- C++ basics (functions, enums, structs)
- Random number generation
- User input/output
- Control structures (loops, conditionals)
- Simple game logic
